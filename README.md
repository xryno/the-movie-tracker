# The Movie Tracker

### About

API project utilising express.js to create endpoints with routers and server side validation.

Additionally, has a login/register function built with passport.js/express-sessions/brcypt.


### Built with

* HTML/CSS/JS
* Express.js
* Passport.js
* Bcrypt.js
* JQuery
* Sqlite
* Sequelize
